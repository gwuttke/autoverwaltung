setTime-Methode (SQLServerCallableStatement)

Legt den angegebenen Parameter auf den angegebenen Zeitwert fest.

Ab SQL Server JDBC Driver 3.0 wird das Verhalten der Methode durch die sendTimeAsDatetime-Verbindungseigenschaft (Festlegen von Verbindungseigenschaften) und SQLServerDataSource.setSendTimeAsDatetime geändert.

Weitere Informationen finden Sie unter Konfigurieren der Art und Weise, wie java.sql.Time-Werte an den Server gesendet werden.
