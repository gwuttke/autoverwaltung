releaseSavepoint-Methode (SQLServerConnection)

Entfernt das angegebene SQLServerSavepoint-Objekt aus der aktuellen Transaktion.

Diese Methode wird von Microsoft JDBC Driver für SQL Server derzeit nicht unterstützt.

Syntax

public void releaseSavepoint(java.sql.Savepoint savepoint)

Parameter

savepoint

Das zu entfernende SavePoint-Objekt.

Ausnahmen

SQLServerException

Hinweise

Diese releaseSavepoint-Methode wird von der releaseSavepoint-Methode in der java.sql.Connection-Schnittstelle angegeben.
