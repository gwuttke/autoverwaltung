Methoden

Name

Beschreibung

free

Mit dieser Methode werden das BLOB sowie die von diesem verwendeten Ressourcen freigegeben.

getBinaryStream

Gibt einen Eingabedatenstrom zum Lesen von Daten aus dem BLOB zurück.

getBytes

Ruft die BLOB-Daten als Bytearray ab.

length

Gibt die Anzahl von Bytes im BLOB zurück.

position

Gibt die Position eines angegebenen Musters im BLOB auf der Grundlage des angegebenen Musters und des Startindexes zurück.

setBinaryStream

Ruft einen Datenstrom ab, mit dem in den BLOB-Wert geschrieben werden kann.

setBytes

Schreibt das angegebene Bytearray ab der angegebenen Position in das BLOB und gibt anschließend die Anzahl der geschriebenen Bytes zurück.

truncate

Kürzt ein BLOB auf die angegebene Länge.
