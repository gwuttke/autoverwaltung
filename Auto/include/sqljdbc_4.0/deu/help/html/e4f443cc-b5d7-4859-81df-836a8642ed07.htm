getTrustServerCertificate-Methode (SQLServerDataSource)

Gibt einen Wert vom Typ Boolean zurück, mit dem angegeben wird, ob die trustServerCertificate-Eigenschaft aktiviert ist.

Syntax

public boolean getTrustServerCertificate()

Rückgabewert

true, wenn "trustServerCertificate" aktiviert ist. Andernfalls wird false verwendet.

Hinweise

Ist die trustServerCertificate-Eigenschaft auf true festgelegt, wird dem SSL-Zertifikat von SQL Server automatisch vertraut, wenn die Kommunikationsschicht mit SSL (Secure Sockets Layer) verschlüsselt ist. Mit anderen Worten, das SQL Server-SSL-Zertifikat wird von Microsoft JDBC Driver für SQL Server nicht überprüft. Der Standardwert ist false.

Ist die trustServerCertificate-Eigenschaft auf false festgelegt, wird das SSL-Zertifikat des Servers von Microsoft JDBC Driver für SQL Server überprüft.
