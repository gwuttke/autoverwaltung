Verwenden von Anweisungen mit SQL

Wenn Sie Daten in einer SQL Server-Datenbank mit Microsoft JDBC Driver für SQL Server und Inline-SQL-Anweisungen verarbeiten, können Sie verschiedenen Klassen verwenden. Die verwendete Klasse hängt vom Typ der SQL-Anweisung ab, die ausgeführt werden soll.

Wenn die SQL-Anweisung keine IN-Parameter enthält, verwenden Sie die SQLServerStatement-Klasse. Wenn IN-Parameter vorhanden sind, verwenden Sie die SQLServerPreparedStatement-Klasse.

Wenn Sie SQL-Anweisungen verwenden müssen, die IN- und OUT-Parameter enthalten, müssen Sie diese als gespeicherte Prozeduren implementieren und mit der SQLServerCallableStatement-Klasse aufrufen. Weitere Informationen zum Verwenden von gespeicherten Prozeduren finden Sie unter Verwenden von Anweisungen mit gespeicherten Prozeduren.
