SSTRANSTIGHTLYCPLD-Feld (SQLServerXAResource)

Hierdurch werden eng verkoppelte XA-Transaktionen ermöglicht, die unterschiedliche XA-Verzweigungstransaktions-IDs (XIDs) aber dieselbe globale Transaktions-ID (GTRID) aufweisen.

Syntax

public static final int SSTRANSTIGHTLYCPLD

Feldwert

Ein Wert vom Typ int (32768).

Hinweise

Jede Transaktion wird anhand einer XA-Verzweigungstransaktions-ID (XID) und einer globalen Transaktions-ID (GTRID) identifiziert. Damit von den Anwendungen eng verkoppelte XA-Transaktionen mit unterschiedlichen XIDs aber den gleichen GTRIDs verwendet werden können, muss SSTRANSTIGHTLYCPLD für den flags-Parameter der XAResource.start-Methode festgelegt werden. Weitere Informationen zum Verwenden dieses Flags finden Sie unter Grundlegendes zu XA-Transaktionen.
