SQLServerClob-Konstruktor (SQLServerConnection, java.lang.String)

Initialisiert eine neue Instanz der SQLServerClob-Klasse, wenn ein SQLServerConnection-Objekt und eine Datenzeichenfolge angegeben werden.

Diese Methode ist seit Version 2.0 von JDBC Driver veraltet. Verwenden Sie stattdessen die createClob-Methode der SQLServerConnection-Klasse.

Syntax

public SQLServerClob(SQLServerConnection connection,
                     java.lang.String data)

Parameter

connection

Ein SQLServerConnection-Objekt.

data

Die CLOB-Daten.
