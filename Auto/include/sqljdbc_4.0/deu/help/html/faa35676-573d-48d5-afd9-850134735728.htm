getSQLXML-Methode (int) (SQLServerResultSet)

Ruft den Wert einer angegebenen Spalte in der aktuellen Zeile des SQLServerResultSet-Objekts als SQLXML-Objekt ab.

Syntax

public final java.sql.SQLXML getSQLXML(int columnIndex)

Parameter

columnIndex

Ein Wert vom Typ int zum Angeben des Spaltenindex.

Rückgabewert

EinSQLXML-Objekt.

Ausnahmen

SQLServerException

Hinweise

Diese getSQLXML-Methode wird von der getSQLXML-Methode in der java.sql.ResultSet-Schnittstelle angegeben.
