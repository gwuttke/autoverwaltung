Methoden

Name

Beschreibung

getParameterClassName

Ruft den vollqualifizierten Namen der Java-Klasse ab, deren Instanzen an die setObject-Methode der SQLServerPreparedStatement-Klasse übergeben werden sollen.

getParameterCount

Ruft die Anzahl von Parametern im SQLServerPreparedStatement-Objekt ab, für die dieses SQLServerParameterMetaData-Objekt Informationen enthält.

getParameterMode

Ruft den Modus des angegebenen Parameters ab.

getParameterType

Ruft den SQL-Typ des angegebenen Parameters ab.

getParameterTypeName

Ruft den datenbankspezifischen Typnamen des angegebenen Parameters ab.

getPrecision

Ruft die Anzahl von Dezimalstellen für den angegebenen Parameter ab.

getScale

Ruft für den angegebenen Parameter die Anzahl von Stellen hinter dem Dezimalzeichen ab.

isNullable

Ruft ab, ob im angegebenen Parameter NULL-Werte zulässig sind.

isSigned

Ruft ab, ob es sich bei den Werten für den angegebenen Parameter um Zahlen mit Vorzeichen handeln kann.
