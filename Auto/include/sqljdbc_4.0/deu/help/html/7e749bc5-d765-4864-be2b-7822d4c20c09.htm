SQLServerDataSource-Elemente

In den folgenden Tabellen sind die Elemente aufgeführt, die von der SQLServerDataSource-Klasse verfügbar gemacht werden.

Konstruktoren

Name

Beschreibung

SQLServerDataSource ()

Initialisiert eine neue Instanz der SQLServerDataSource-Klasse.

Felder

Keine

Geerbte Felder

Keine

Methoden

Name

Beschreibung

getApplicationIntent

Gibt den Wert der applicationIntent-Verbindungseigenschaft zurück.

getApplicationName

Gibt den Anwendungsnamen zurück.

getConnection

Stellt eine Verbindung mit der Datenquelle her, für die dieses SQLServerDataSource-Objekt steht.

getDatabaseName

Gibt den Datenbanknamen zurück.

getEncrypt

Gibt einen Wert vom Typ Boolean zurück, mit dem angegeben wird, ob die encrypt-Eigenschaft aktiviert ist.

getDescription

Gibt eine Beschreibung der Datenquelle zurück.

getFailoverPartner

Gibt den Namen des Failoverservers zurück, der in einer Datenbankspiegelungskonfiguration verwendet wird.

getHostNameInCertificate

Gibt den Hostnamen zurück, der bei der Überprüfung des Secure Sockets Layer (SSL)-Zertifikats von SQL Server verwendet wird.

getInstanceName

Gibt den Namen der SQL Server-Instanz zurück.

getLastUpdateCount

Gibt einen Wert vom Typ boolean zurück, mit dem angegeben wird, ob die lastUpdateCount-Eigenschaft aktiviert ist.

getLockTimeout

Gibt einen Wert vom Typ int zurück, mit dem angegeben wird, wie lange von der Datenbank bis zum Melden eines Sperrtimeouts gewartet wird (in Millisekunden).

getLoginTimeout

Gibt die Anzahl von Sekunden zurück, die vom SQLServerDataSource-Objekt bei der Verbindungsherstellung gewartet wird.

getLogWriter

Gibt einen Zeichenausgabedatenstrom für alle Protokollierungs- und Ablaufverfolgungsmeldungen zurück.

getMultiSubnetFailover

Gibt den Wert der multiSubnetFailover-Verbindungseigenschaft zurück.

getPacketSize

Gibt die aktuelle Netzwerkpaketgröße (in Bytes) für die Kommunikation mit SQL Server zurück.

getPortNumber

Gibt die aktuelle Portnummer für die Kommunikation mit SQL Server zurück.

getReference

Gibt einen Verweis auf dieses SQLServerDataSource-Objekt zurück.

getResponseBuffering

Gibt den Antwortpuffermodus für dieses SQLServerDataSource-Objekt zurück.

getSelectMethod

Gibt den Standardcursortyp zurück, der für alle Resultsets verwendet wird, die mithilfe dieses SQLServerDataSource-Objekts erstellt werden.

getSendStringParametersAsUnicode

Gibt einen Wert vom Typ Boolean zurück, mit dem angegeben wird, ob das Senden von Zeichenfolgenparametern an den Server im Unicode-Format aktiviert ist.

getSendTimeAsDatetime

Sie gibt die Einstellung der SendTimeAsDatetime-Verbindungseigenschaft zurück.

getServerName

Gibt den Namen des Computers zurück, auf dem SQL Server ausgeführt wird.

getTrustServerCertificate

Gibt einen Wert vom Typ Boolean zurück, mit dem angegeben wird, ob die trustServerCertificate-Eigenschaft aktiviert ist.

getTrustStore

Gibt den Pfad (einschließlich Dateiname) zur trustStore-Zertifikatsdatei zurück.

getURL

Gibt die URL zurück, die zum Herstellen einer Verbindung mit der Datenquelle verwendet wird.

getUser

Gibt den Benutzernamen zurück, der zum Herstellen einer Verbindung mit der Datenquelle verwendet wird.

getUseSQLServerBaseDate

Sie gibt die Einstellung der useSQLServerBaseDate-Verbindungseigenschaft zurück.

getWorkstationID

Gibt den Namen des Clientcomputers zurück, der zum Herstellen einer Verbindung mit der Datenquelle verwendet wird.

getXopenStates

Gibt einen Wert vom Typ Boolean zurück, mit dem angegeben wird, ob das Konvertieren von SQL-Status in XOPEN-kompatible Status aktiviert ist.

isWrapperFor

Gibt an, ob es sich bei diesem Datenquellenobjekt um einen Wrapper für die angegebene Schnittstelle handelt.

setApplicationIntent

Legt den Wert der applicationIntent-Verbindungseigenschaft fest.

setApplicationName

Legt den Anwendungsnamen fest.

setAuthenticationScheme

Gibt die Art der integrierten Sicherheit an, die für die Anwendung verwendet werden soll.

setDatabaseName

Legt den Namen der Datenbank fest, mit der eine Verbindung hergestellt werden soll.

setDescription

Legt die Beschreibung der Datenquelle fest.

setEncrypt

Legt einen Wert vom Typ Boolean fest, mit dem angegeben wird, ob die encrypt-Eigenschaft aktiviert ist.

setFailoverPartner

Legt den Namen des Failoverservers fest, der in einer Datenbankspiegelungskonfiguration verwendet wird.

setHostNameInCertificate

Legt den Hostnamen fest, der bei der Überprüfung des Secure Sockets Layer (SSL)-Zertifikats von SQL Server verwendet werden soll.

setInstanceName

Legt den Namen der SQL Server-Instanz fest.

setIntegratedSecurity

Legt einen Wert vom Typ Boolean fest, mit dem angegeben wird, ob die integratedSecurity-Eigenschaft aktiviert ist.

setLastUpdateCount

Legt einen Wert vom Typ Boolean fest, mit dem angegeben wird, ob die lastUpdateCount-Eigenschaft aktiviert ist.

setLockTimeout

Legt einen Wert vom Typ int fest, mit dem angegeben wird, wie lange von der Datenbank bis zum Melden eines Sperrtimeouts gewartet wird (in Millisekunden).

setLoginTimeout

Legt die Anzahl von Sekunden fest, die vom SQLServerDataSource-Objekt bei der Verbindungsherstellung gewartet wird.

setLogWriter

Legt einen Zeichenausgabedatenstrom für alle Protokollierungs- und Ablaufverfolgungsmeldungen fest.

setMultiSubnetFailover

Legt den Wert der multiSubnetFailover-Verbindungseigenschaft fest.

setPacketSize

Legt die aktuelle Netzwerkpaketgröße (in Bytes) für die Kommunikation mit SQL Server fest.

setPassword

Legt das Kennwort fest, das für die Verbindung mit SQL Server verwendet wird.

setPortNumber

Legt die Portnummer für die Kommunikation mit SQL Server fest.

setResponseBuffering

Legt den Antwortpuffermodus für Verbindungen fest, die unter Verwendung dieses SQLServerDataSource-Objekts erstellt werden.

setSelectMethod

Legt den Standardcursortyp fest, der für alle Resultsets verwendet wird, die mithilfe dieses SQLServerDataSource-Objekts erstellt werden.

setSendStringParametersAsUnicode

Legt einen Wert vom Typ Boolean fest, mit dem angegeben wird, ob das Senden von Zeichenfolgenparametern an den Server im Unicode-Format aktiviert ist.

setSendTimeAsDatetime

Gibt die Art und Weise an, wie java.sql.Time-Werte an den Server gesendet werden.

setServerName

Legt den Namen des Computers fest, auf dem SQL Server ausgeführt wird.

setTrustServerCertificate

Legt einen Wert vom Typ Boolean fest, mit dem angegeben wird, ob die trustServerCertificate-Eigenschaft aktiviert ist.

setTrustStore

Legt den Pfad (einschließlich Dateiname) zur trustStore-Zertifikatsdatei fest.

setTrustStorePassword

Legt das Kennwort fest, das zum Überprüfen der Integrität der trustStore-Daten verwendet wird.

setURL

Legt die URL fest, die zum Herstellen einer Verbindung mit der Datenquelle verwendet wird.

setUser

Legt den Benutzernamen fest, der zum Herstellen einer Verbindung mit der Datenquelle verwendet wird.

setWorkstationID

Legt den Namen des Clientcomputers fest, der zum Herstellen einer Verbindung mit der Datenquelle verwendet wird.

setXopenStates

Legt einen Wert vom Typ Boolean fest, mit dem angegeben wird, ob das Konvertieren von SQL-Status in XOPEN-kompatible Status aktiviert ist.

unwrap

Gibt ein Objekt zurück, das die angegebene Schnittstelle implementiert, um den Zugriff auf die Microsoft JDBC Driver für SQL Server-spezifischen Methoden zu ermöglichen.
