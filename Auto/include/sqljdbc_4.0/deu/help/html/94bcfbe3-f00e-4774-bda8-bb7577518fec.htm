Verbinden von SQL Server mit dem JDBC-Treiber

Eine der grundlegenden Aufgaben, die mit Microsoft JDBC Driver für SQL Server ausgeführt werden, ist das Herstellen einer Verbindung mit einer SQL Server-Datenbank. Die gesamte Interaktion mit der Datenbank erfolgt über das SQLServerConnection-Objekt. Da der JDBC-Treiber eine extrem flache Architektur aufweist, beziehen sich nahezu alle wesentlichen Verhalten auf das SQLServerConnection-Objekt.

Wenn SQL Server nur einen IPv6-Port überwacht, legen Sie die Systemeigenschaft "java.net.preferIPv6Addresses" fest, um sicherzustellen, dass für die Verbindung mit SQL Server IPv6 statt IPv4 verwendet wird:

System.setProperty("java.net.preferIPv6Addresses", "true");

Die Themen in diesem Abschnitt beschreiben, wie Sie Verbindungen mit einer SQL Server-Datenbank herstellen und verwalten.
