isClosed-Methode (SQLServerResultSet)

Gibt an, ob dieses SQLServerResultSet-Objekt geschlossen wurde.

Syntax

public boolean isClosed()

Rückgabewert

true, wenn dieses SQLServerResultSet-Objekt geschlossen ist. false, wenn es noch geöffnet ist.

Ausnahmen

SQLServerException

Hinweise

Diese isClosed-Methode wird von der isClosed-Methode in der java.sql.ResultSet-Schnittstelle angegeben.
