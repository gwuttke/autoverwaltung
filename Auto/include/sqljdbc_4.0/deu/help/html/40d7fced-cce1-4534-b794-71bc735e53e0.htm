updateDateTimeOffset (SQLServerResultSet)

Diese Methode wurde in Microsoft SQL Server JDBC Driver 3.0 hinzugefügt.

Aktualisiert den Wert der Spalte, die für einen DateTimeOffset-Klasse-Wert angegeben wurde.

Überladungsliste

Name

Beschreibung

updateDateTimeOffset(int, microsoft.sql.DateTimeOffset)

Aktualisiert bei einer vorhandenen nullbasierten Spaltenordinalzahl die für den DateTimeOffset-Klasse-Wert festgelegte Spalte.

updateDateTimeOffset(string, microsoft.sql.DateTimeOffset)

Aktualisiert bei einem vorhandenen Spaltennamen die für den DateTimeOffset-Klasse-Wert festgelegte Spalte.
