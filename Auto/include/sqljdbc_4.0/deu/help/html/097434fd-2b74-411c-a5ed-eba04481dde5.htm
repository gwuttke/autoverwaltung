SQLServerDataSource-Klasse

Stellt eine Liste mit spezifischen Eigenschaften für das Herstellen einer Verbindung mit einer SQL Server-Datenbank (unter Verwendung eines SQLServerConnection-Objekts) dar.

Paket: com.microsoft.sqlserver.jdbc

Implementiert:ISQLServerDataSource, DataSource, java.io.Serializable, javax.naming.Referenceable

Syntax

public class SQLServerDatabaseMetaData

Hinweise

Diese Klasse unterstützt das Entpacken in die Klasse SQLServerDataSource, die Schnittstelle ISQLServerDataSource und die Schnittstelle DataSource. Weitere Informationen finden Sie unter Wrapper und Schnittstellen.
