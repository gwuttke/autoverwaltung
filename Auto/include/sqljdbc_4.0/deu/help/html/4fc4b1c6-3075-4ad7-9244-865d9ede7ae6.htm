Arbeiten mit Resultsets

Bei der Verarbeitung von Daten in einer SQL Server-Datenbank besteht eine Methode zum Bearbeiten der Daten darin, ein Resultset zu verwenden. Microsoft JDBC Driver für SQL Server unterstützt über das SQLServerResultSet-Objekt die Verwendung von Resultsets. Mithilfe des SQLServerResultSet-Objekts können Sie die von einer SQL-Anweisung oder gespeicherten Prozedur zurückgegebenen Daten abrufen, die Daten bei Bedarf aktualisieren und dann wieder in der Datenbank speichern. 

Das SQLServerResultSet-Objekt enthält darüber hinaus Methoden, um in den Datenzeilen zu navigieren, die enthaltenen Daten abzurufen oder festzulegen und die Sensitivität gegenüber Änderungen in der zugrunde liegenden Datenbank einzurichten.

Weitere Informationen zur Verwaltung von Resultsets, einschließlich deren Sensitivität gegenüber Änderungen, finden Sie unter Verwalten von Resultsets mit dem JDBC-Treiber.

Die Themen in diesem Abschnitt beschreiben verschiedene Möglichkeiten, wie Sie mithilfe von Resultsets die in einer SQL Server-Datenbank enthaltenen Daten bearbeiten können.
