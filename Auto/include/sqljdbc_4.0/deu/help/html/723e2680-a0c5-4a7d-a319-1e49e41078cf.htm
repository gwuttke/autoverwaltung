Diagnostizieren von Problemen mit dem JDBC-Treiber

Unabhängig davon, wie sorgfältig eine Anwendung entworfen und entwickelt wird, werden unvermeidlich Probleme auftreten. In diesem Fall ist es wichtig, über Verfahren zum Diagnostizieren dieser Probleme zu verfügen. Bei der Verwendung von Microsoft JDBC Driver für SQL Server sind häufige Probleme das Fehlen der richtigen Treiberversion und Fehler beim Herstellen einer Verbindung mit einer Datenbank.

In den Themen in diesem Abschnitt werden unterschiedliche Methoden zum Diagnostizieren dieser und anderer Probleme beschrieben, unter anderem Fehlerbehandlung, Überprüfen der Treiberversion, Ablaufverfolgung und Problembehebung bei Verbindungsproblemen.
